# TechStore Frontend

This folder contains the React/Vite storefront and admin UI for TechStore.

For the complete project overview, setup guide, API map, demo accounts, and resume-ready summary, see the root [README.md](../README.md).

## Local Development

```powershell
npm install
Copy-Item .env.example .env
npm run dev
```

Set `VITE_API_URL=http://localhost:5000/api` in `.env` when running the backend locally.
